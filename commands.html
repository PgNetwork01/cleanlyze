Data Cleaning

  • remove_duplicates(df): Removes duplicate rows from a DataFrame.
  • handle_missing_values(df, strategy='mean'): Handles missing values with a specified strategy.

Data Transformation

  • normalize_data(df): Normalizes the data using Min-Max scaling.
  • create_polynomial_features(df, degree=2): Generates polynomial features.

Data Visualization

  • plot_histogram(df, column_name): Plots a histogram for a given column.
  • plot_correlation_matrix(df): Plots a correlation matrix heatmap.
