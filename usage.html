Installation

Install Cleanlyze using pip:

pip install cleanlyze

Basic Example


import cleanlyze as cz
import pandas as pd

# Example DataFrame
data = {'name': ['Alice', 'Bob', 'Alice'], 'score': [85, 90, 85]}
df = pd.DataFrame(data)

# Remove duplicates
clean_df = cz.remove_duplicates(df)
print(clean_df)
